# Camions BL — PMO & Innovation Dashboard

Outil de gestion de portefeuille projets pour Camions BL.

## Fonctionnalités
- Processus PMO pré-Gate 0 en 5 étapes
- Triage conversationnel des idées (projet, amélioration, opération, rejet)
- Scoring pondéré sur 6 critères stratégiques
- Gestion des ressources nommées avec disponibilité h/semaine
- Affectation des personnes aux projets
- Analyse de capacité avec identification des goulots
- Rapport Gate 0 pour approbation exécutive

## Déploiement
Site statique — aucune base de données requise. Les données sont sauvegardées dans le localStorage du navigateur.

```bash
# Vercel
vercel --prod
```
